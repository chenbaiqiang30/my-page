种草四步飞轮

捕捉、理解、放大、激发——让用户成为你最好的推销员

1

捕捉

从用户的个体表达中捕捉新机会。翻热词热帖、找独特需求、看上游搜索、聊超级用户，发现产品创新和营销突破口。

2

理解

理解用户真正向往的生活方式。不看人口学标签，而是深入生活场景，找到用户表达背后的深层动机和情绪价值。

3

放大

又快又准地将机会转化为收益。采用反漏斗策略，让用户参与所有环节，从核心人群逐层扩圈，精准触达目标用户。

4

激发

验证用户是否真正被激发。观察用户的购买、复购、分享行为，确保种草成功，形成自发传播的正向循环。

种草的本质不是企业自夸"产品多厉害"，而是源于真实用户的表达"我用着怎么样"。

01

导论：成功种草的方法论

什么是种草？

当你看到其他人分享的好东西，或看到品牌发布的广告后特别想买，你就是"被种草"了。种草的核心是源于真实用户的表达，而非企业单向输出。

用户的购买决策路径已经改变：以前看品牌广告就下单，现在即便受品牌影响，下单前也会先去小红书做功课，看其他用户的真实反馈。

经典案例：肯德基疯狂星期四

现象：从2018年的促销活动演变为全网狂欢的"疯四文学"，每周四搜索流量达到其他时间的6倍以上。

结果：每周四销售额比其他工作日高约40%，且大量免费流量来自用户自发创作和传播。

启示：真正的种草成功是用户被激发后主动为品牌发声，形成自传播效应。

种草成功的标志

短期标志
  • 内容激发了用户购买欲，转化率显著提升
  • 大量用户主动购买并为产品发声
  • 用户在品牌内容下给出正面评论
  • 用户主动发帖分享使用方法和感受
长期标志
  • 获客成本越来越低
  • 用户分享的内容转化效果超过品牌广告
  • 形成持续的口碑传播和自然增长

种草四步缺一不可，头尾相连，最终形成飞轮。当飞轮越转越顺，你就能成为一个种草型企业。

02

捕捉：怎样从用户表达中发现新机会？

为什么要用小红书捕捉？

相比传统调研，小红书能让你用相对更高效的方式获得用户更真实的声音。用户在小红书上发笔记、做分享，内容真实度更高，且不受调研框架限制。

传统调研效率太低。过去宝洁要花几个月走访用户发现蓝色染料漂白的需求，今天在小红书上搜索就能立刻找到大量用户分享。

四个高效捕捉方法

方法一：翻热词热帖

定义：找到点赞、评论、收藏很高，且意料之外的内容。

案例：乐高积木发现用户把"梵高星月夜积木"当作家居装饰品，而非玩具。用户说"摆在柜子上就是一幅很美的装饰品"，这个意外需求帮助乐高开拓了家居软装市场。

操作要点：热度不是越高越好，要捕捉那些意料之外又有热度的内容，既满足创新，又避免陷入极其小众的特殊需求。

方法二：找独特需求

定义：观察目标用户区别于其他用户群体的独特需求。

案例：林氏家居针对26-30岁年轻人，发现他们的长尾搜索词中出现"可移动"。深挖后理解这代表"租房随时跑路"和"一物多用"的需求，最终推出"包包椅"成为销量增速最快的产品。

操作要点：①锁定人群 ②看长尾搜索词（前几十个词可能都是大众需求）③回归用户表达理解具体场景。

方法三：看上游搜索

定义：看用户在搜索你的产品或品牌之前，在搜索什么内容。

案例：凯乐石发现用户在搜索冲锋衣前，会先搜索"爬泰山攻略"等旅行目的地。于是针对不同山峰的环境特征，推出性能最匹配的产品，营销无比丝滑，获客成本下降近一半。

操作要点：上游搜索往往隐藏着具体的使用场景，包含时间、地点、人物、需求，能帮你找到精准的场景营销突破口。

方法四：聊超级用户

定义：找到在某个领域内比普通消费者更专业、要求更高的"用户专家"。

筛选标准：①粉丝约1000左右（有代表性但不是KOL）②大量分享相关产品，尤其只在细分类目做分享。

案例：babycare找那些大量测评纸尿裤的用心养娃妈妈，虽然她们不是母婴达人，但对产品理解比一般用户专业，能输出有效建议。

操作要点：选择粉丝量千人左右的用户，与他们长期联系，倾听建议并真诚反馈。

捕捉是整个种草行动的基础。如果缺少捕捉，一味按自己的偏好走，很容易陷入充分竞争，失去创新的机会。

成功案例：源制所蓝莓汁

从捕捉到产品上市的完整路径

捕捉机会：通过小红书笔记数量发现"蓝莓"是用户认知中最健康的水果，用户甚至能说出"花青素"等营养成分。

发现问题：用户吐槽"果汁=糖水"，担心糖分高、防腐剂多、营养成分低、配料表不干净。

产品调整：①舍弃糖，用混合果蔬汁调整口感 ②使用NFC技术保留营养 ③配料表只有水、蓝莓汁和各种果汁。

包装创新：设计成200ml小瓶，标注"26颗蓝莓"，配料表干净，让用户感知健康。

结果：大量用户自发分享"好爱它的配料表""不太甜""健康"，成功打开市场。

成功案例：凡士林雪融霜

从功能营销到情绪价值的转变

初期困境：围绕"保湿"功能找达人铺笔记，但保湿是所有面霜的基础功能，营销效果不理想。

捕捉转机：通过小红书后台发现购买用户最感兴趣的内容是"自我接纳&个人成长"。

策略调整：转而与情感博主合作，讲述"把男友踢出局后与婆婆成闺蜜""离婚后做清醒独立女人"的故事，产品自然融入场景。

结果：单条视频获得96万点赞，上市100天后成为某电商平台保湿护理类热销第一。

启示：产品越来越近似时，必须挖掘情绪价值，这才是品牌营销差异化的抓手。

03

理解：怎样解码用户的真实向往？

为什么不能只看标签？

传统的人口学标签（年龄、性别、地域、职业）越来越不好用。因为随着生活选择越来越丰富，同一标签下的用户需求可能完全对立。

30-40岁都市白领女性，有人希望养生壶功能简单，有人希望功能复杂有仪式感。同一个宿舍的4个女大学生，对隐形眼镜的需求也截然不同。

理解生活方式的核心

生活方式是用户在日常生活中形成的一系列行为模式、消费习惯、价值观念的总和。理解用户向往的生活方式，才能找到真正有穿透力的创新方向。

案例：思念柿柿如意汤圆

表层需求：用户吐槽汤圆含反式脂肪酸、不健康、口味新奇等。

深层理解：元宵节用户真正向往的不只是好吃，而是通过各种方式表达对美好未来的祝福和期待。

创新方向：结合"柿柿如意"谐音梗和红柿子造型，既满足仪式感，又让祝福更有趣、更直观。

结果：上市三个月销售额破亿，大约每5个吃汤圆的家庭就有1家吃"柿柿如意"。大量用户争相分享如何煮出完美造型。

三个理解方法

方法一：串联多元行为

案例：科沃斯扫地机器人

用户吐槽："扫地机器人难看"——但不是产品本身难看，而是摆在洗手盆下方不协调、线路乱七八糟破坏家装美感。

行为串联：通过灵犀后台发现，正在选购扫地机器人的用户高频搜索词都与"家装"有关，处于软装阶段。

深层向往：用户向往的是"一进门就能感觉治愈的家"，扫地机器人需要融入家装风格。

产品调整：推出奶油风扫地机器人——低饱和浅色系、线条圆润、电线藏进机身。

结果：上市当季度销量过亿，成为行业搜索量第一的单品。

方法二：找到现状与向往的差距

案例：Usmile笑容加儿童电动牙刷Q20

用户需求：刷头薄一点、造型有趣、方便清洁等表层需求。

线下观察：到口腔医院观察发现，孩子们普遍不会刷牙、没有耐心刷够3分钟。

深层向往：家长向往"孩子口腔健康、不长蛀牙"，现状是"孩子不会刷、没耐心"。

产品设计：①骨传导发声播放儿歌 ②重力感应提示刷哪个区域 ③刷完一个区域鼓励"你真棒"。

结果：上市几周冲到天猫电动牙刷榜第一，占领儿童电动牙刷50%+市场份额。

方法三：挖掘情感表达

案例：熊猫布布吸奶器

线下反馈：用户希望"不疼地吸干净"、便于移动。

线上捕捉：小红书上热门话题"凌晨四点的妈妈"——半夜喂奶/吸奶时整个城市漆黑寂静，老公在睡觉，不敢开灯怕吵醒孩子。

情感需求：凌晨的妈妈需要陪伴感。

产品创新：添加呼吸小夜灯，像呼吸一样闪烁，给凌晨的妈妈带来安静又忠实的陪伴。

启示：呼吸小夜灯成本不高、技术不难，难的是理解到用户对陪伴的向往。情绪价值的满足常常在这样"多做一步"的动作里。

理解人的能力需要三个要素：看过更广阔的世界、深入研究过自己、直接接触过足够多的人。

理解用户的操作要点
  • 不要孤立地看待用户的单一诉求，而要串联起生活中的多元行为
  • 不要只停留于需求的表象，而是要找到表象和向往之间的差距或矛盾点
  • 不要只关注产品本身，而要找到用户在使用产品过程中的情感表达
  • 线上线下相结合，互相佐证，理解用户需求背后的向往和深层动机

04

放大：怎样把机会精准快速变现？

放大的核心：快且准

把握住新机会后，需要通过产品、营销、服务等一系列动作，在更大范围内验证用户是否会被激发。速度要快，否则被竞争对手抢先；方向要准，否则浪费资源且无法转化。

某国货美妆品牌最早发现液体修容机会，但因放大速度不够快，不到半年就被各大品牌跟进，失去了自己开拓出来的市场。

原则一：反漏斗营销

传统正漏斗问题：广而告之→曝光越多成本越高→命中大量非目标用户→差评累积→营销越来越难。

反漏斗策略：从核心人群逐层扩圈

第一步：识别核心人群（最有可能被打动、转化率最高的用户），用适配内容精准触达。

第二步：充分渗透核心人群后，找其他高潜人群（已购买且评价好的群体），逐层扩散。

第三步：被种草的用户形成自发口碑扩散，进一步影响更多人群。

案例：亚朵星球深睡枕

核心人群：有睡眠障碍的人（失眠、夜里易醒）→ 强调"一觉睡到天亮"→ 销量和口碑快速起来。

第一次扩圈：科学养生人群 → 强调"保护颈椎"→ 成功吸引这类用户。

第二次扩圈：新手妈妈 → 强调"碎片化时间快速进入深度睡眠"→ 进一步扩大市场。

操作要点：①选择扩圈人群时，看具体购买理由是否能代表该人群真实需求 ②为新人群针对性设计营销策略（人群-痛点-内容-卖点结合）。

结果：2023年3月上市，当年双11冲到各大平台枕头品类销量榜首，截至2025年4月累计销量突破600万只。

原则二：让用户参与所有环节

以往品牌让用户参与头尾两端（研发前调研、上市后收集反馈），但这样容易导致产品和营销与用户需求脱节。更好的做法是让用户参与过程中的每一步。

案例：方仔照相馆

产品设计阶段：给用户三次免费修改机会，积累设计手册，提高一稿过概率。

放大过程-包装问题：用户反馈拼好的人偶装不回盒子 → 重新设计包装 → 晒单用户数显著提升。

放大过程-造型问题：发现女性用户比例低于预期 → 用户反馈方脸不如圆脸可爱 → 升级成嘟嘟圆脸 → 送礼场景下女性用户大幅增加。

放大过程-内容调整：观察用户晒单偏好 → 发现用户把人偶作为家居摆件晒 → 营销内容加入场景细节 → 点击率和进店率更高。

启示：在放大的每个环节都让用户参与，及时根据反馈调整产品、卖点和营销方向。

放大过程中的三大典型问题

问题一：怎么选品？
  • 核心原则：不要直接选当下销量最高的品，它不一定适配小红书的调性
  • 正确做法：回归用户的个体表达，重新捕捉和理解，选出在小红书上用户最有需求的品
  • 案例：韩餐店线下销量最高是石锅拌饭，但小红书上生腌酱蟹（因食材新鲜、秘制酱料、拍照好看）吸引了大量远距离用户
  • 宜家线下卖得好的是大件家具，但小红书上小件家具（方格柜、洞洞小推车）更受欢迎
问题二：怎么选达人？
  • 核心原则：优先看目标用户在达人粉丝中的占比，而非粉丝量级
  • 案例：嗨小瓶运动咖啡 → 不选咖啡达人（粉丝追求口感）→ 不选运动员（粉丝觉得专业产品不适合自己）→ 选医美达人（粉丝是对身材有极高要求、愿意为精致付费的女性）→ 单条视频投产比超10倍
  • 内容合作原则：不要强制达人按剧本走，尊重达人和粉丝的沟通方式。多支持，少限制。先让达人自由发挥，积累案例后总结规律，再复制给更多达人
  • 案例：羊织道 → 达人自由创作 → 复盘发现关键要素（动态拍摄、单反设备）→ 固化要求 → 为达人提供丝巾、选择拱门场景 → 降低创作门槛
问题三：怎么铺笔记？
  • 核心任务：品牌主要发力消除用户的购买卡点
  • 两大卡点：①对产品还有疑问（适不适合自己、怎么使用效果最佳）②看到其他用户的差评，担心产品不够好
  • 操作方法：①加热现有正面笔记，覆盖用户搜索的问题 ②发新笔记解答未被覆盖的高搜索量问题（不一定是官方发，可以是达人或用户）③分类处理差评：误会→解释清楚；非目标用户→不用在意；大量目标用户吐槽同一问题→改产品改服务
  • KFS模型：K（选对达人）+ F（做好内容并投放）+ S（覆盖用户搜索）= 营销效果稳定

放大的关键是人群、需求、产品、卖点、媒介、内容的匹配。只有全部适配，才能保证营销成功。

05

激发：为什么激发是种草的关键一步？

什么是激发态？

激发态是用户因为看到营销内容或体验到产品，觉得真的命中了自己的需求、体验超出预期，因此自发地购买、复购、为品牌发声传播。具体行为是点赞、评论、转发、复购。

激发既是种草的起点，也是种草的终点。捕捉就是从用户被激发的现象开始，而种草的最终目标是让目标用户普遍被激发。

假的激发态：两种常见情况

情况一：销量上涨≠种草成功

销量上涨可能只是因为营销预算充足。停止投放或降低投放后，如果销量完全等比例回落，说明用户没有被真正激发。

案例：儿童防晒喷雾 → 内容互动数据很高，但用户自发分享很少，有限的分享都是差评（喷雾颗粒宝宝容易吸入、容易刺激眼睛）→ 差评累积后，即便广告触达新用户，看到满屏差评也不敢下单 → 营销越来越难。

情况二：内容互动好≠种草成功

用户愿意为内容点赞、评论，可能是对内容有好感，但对内容的好感不一定导向购买产品。

案例：成人英语学习产品 → ROI太低 → 以为小红书用户需求是应试 → 放弃小红书。实际上产品没问题，问题出在达人选择上：选了考研考学达人，但真正的目标用户是40岁以上、为自我提升和海外生活准备的人群。找到长期居住海外的读书类达人后，效果立刻改善。

没有达到激发态怎么办？

不要只看ROI。ROI是结果指标，只看数字无法定位问题出在哪个环节。应该把激发态贯穿始终，在每个环节都回归用户做验证。

案例：babycare的验证机制

产品概念阶段：开评审会给内部忠实用户（既是员工又是产品用户），产品经理只有3分钟讲方案，打动用户才能通过评审。

产品设计阶段：找内部用户看设计，判断外观、功能是否符合使用习惯，有问题当场改。

样品试用阶段：让评审者试用，给出真实使用反馈。如"婴儿水杯吸管容易漏"，立刻调整结构。

小范围上架：先在天猫店小范围上架，不投流、纯靠自然流量测试：看谁会买、复购率如何、评价里有哪些需求。同时邀请小红书母婴博主（超级用户）试用。

营销阶段：小钱测试不同内容，互动量、转化率高的内容再追加预算放大。

结果：每个环节通过用户反馈校准方向，确保用户达到激发态，爆品概率自然更高。

三个常见问题自查清单

问题一：目标用户找对了吗？
  • 现象：用户被激发，但没有表现在销售上（叫好不叫座）
  • 案例：美妆品牌做"为化妆减负"的产品，找超级美妆用户测试 → 她们喜欢但反馈想要"独特质地和颜色" → 与"减负"理念冲突 → 重新找"时间有限、想快速化妆上班"的人群，才找到真正的目标用户
  • 解决方法：通过替代产品找目标用户。如元气森林要替代矿泉水，就研究日常以矿泉水为主要饮水来源的用户；要替代冰红茶，就研究传统冰红茶用户的痛点
  • 处理原则：①目标用户找错了 → 重新寻找 ②非目标用户被激发了 → 尽量坚持原方向，如果一定要转换，重做种草全链路
问题二：用户的反馈是真的吗？
  • 核心问题：用户嘴上说一切都好，但掏钱时不想买
  • 案例：调研拍立得颜色偏好 → 用户说喜欢蓝色、粉色、淡紫色 → 调研结束让用户任选一个带走 → 绝大多数选白色（因为白色百搭）
  • 解决方法：观察用户的无意识反应（身体前倾、眼睛发亮、迫不及待询问、主动交定金、主动介绍给身边人）
  • 最佳实践：回到实际使用场景观察。如厨具品牌让调研人员在午餐时间上门，实际看用户怎么炒菜；亚朵星球在酒店开20个房间，让用户实际睡一觉后反馈，统计正面/负面感受词的数量和比例
  • 关键：用户在描述亲身感受时的用词和无意识反应（体态、眼神、语速）骗不了人
问题三：你尊重用户真实的激发态了吗？
  • 核心问题：用户给出真实反馈，但与研发初心不一致，听不听用户的？
  • 答案：听用户的！
  • 案例1：Bobbi Brown妆前柔润底霜 → 用户起外号"橘子面霜"（因颜色和气味像橘子）→ 品牌接受外号 → 成为全网现象级爆款
  • 案例2：剃须刀礼盒 → 品牌得意大礼盒和赠品 → 用户差评：赠品用不上、盒子太大占地方、不如降价实惠 → 听用户的才能让用户真正喜欢
  • 案例3：开放式耳机 → 品牌花精力改善播放效果 → 用户把它当配饰，更在意造型和颜色 → 听用户意见做成好看配饰的品牌赢得市场
  • 小红书"听劝"文化：愿意倾听用户意见、按用户意见改变的品牌，用户也会帮助它更快成功

当你同时做到找对目标用户、观察真实反馈、尊重用户意见，用户会更积极地分享、讨论、持续购买。这些被激发的用户，会帮助你种草顺利。

激发态贯穿始终

激发不仅是最后一步，而应贯穿捕捉、理解、放大的所有环节。如果某个环节用户没达到激发态，最好不要推进下一步。

飞轮效应：当用户进入激发状态 → 持续购买、正面口碑 → 成为品牌资产 → 生意越做越容易 → 种草越做越轻松 → 更多素材可捕捉 → 更多用户愿意建立联系 → 验证想法更容易 → 飞轮快速转动。

案例：种草飞轮怎样转动？

案例一：RED CHAMBER多用膏

捕捉与理解：找到品牌方向

捕捉：小红书上用户讨论越来越多的不是"哪些彩妆颜色好看"，而是"哪些彩妆成分不健康"。

理解：对成分的关注背后是对健康的要求，用户向往配料表更干净、去除危险成分的彩妆。

品牌定位：做"纯净彩妆"品牌（当时中国尚未出现）。

产品选择：捕捉到用户抱怨"希望赖床不想早起化妆" → 理解为"化妆减负"的需求 → 选择多用膏（一膏多用：嘴唇、脸颊、眼周，减少产品数量和化妆时间）。

放大踩坑：达人选择错误

初期策略：找彩妆头部达人带货 → 销量有限、投产比低。

问题分析：彩妆达人的粉丝把化妆当作愉悦过程，愿意尝试更多颜色、接受更长化妆时间，不是多用膏的目标用户（时间紧张、希望减负的人群）。

困境时长：2年找不到营销破局方向。

破局：重新捕捉理解

关键笔记：一位2000粉丝用户的笔记获得3万赞 → 两张图：①全脸点涂橘色多用膏 ②晕染开的裸妆效果 → 标题四个字：裸门永存！

理解：追求裸妆风格的用户更可能接受脸上用更少颜色；裸妆更简单、妆感更少，符合希望省时又美感的用户诉求；"先点再涂"两张图直观展示操作简单。

策略调整：①找适合裸妆、展示裸妆的达人（未必是彩妆达人）②主打裸妆 ③展示"先点再涂"操作过程 ④展示最终裸妆妆容。

结果：这一款多用膏每年在小红书上实现上亿销售额，赶上头部水平。

RED CHAMBER的启示
  • 初创品牌忘掉"教育市场"，学会"回归用户"。品牌价值不在于创造新需求，而是成为用户现有需求的最佳解决方案
  • 放大遇到困境时，用捕捉和理解去突破。遇到困难不可怕，回到用户常常能找到答案
  • 真正该借的"势"，恰恰是用户的力量

案例二：宜家

线下惯性vs线上现实

线下优势：大件家具（沙发、床、床垫、柜子）撑起主要销量。

初期思路：大件卖得好且贵，更容易覆盖营销成本 → 计划在小红书上卖大件。

意外发现：线下积压的"波席当移动边桌"（可移动可升降小边桌）在小红书突然大卖 → 博主视频展示多场景使用（床上餐桌、矮茶几、客厅餐桌、厨房操作台）→ 8万点赞。

理解：波席当的价值在可移动、可升降的灵活性（"一块砖，哪里需要哪里搬"）→ 线下静态陈列无法呈现，线上短视频完美展示切换场景的能力 → 视频媒介与产品卖点适配。

捕捉：小件才是优势

用户分享最多：不是大件家具，而是小件产品（波席当、不锈钢大盆、办公收纳盒、洞洞小推车、小灯笼）。

竞争格局分析：①大件家具：沙发（顾家）、床垫（喜临门）有专注品牌，技术积累和品牌积累更强 ②小件家具：宜家有审美设计能力和丰富产品组合，用户认可度高，愿意为设计支付溢价。

客单价问题：小件便宜（200的灯vs2000的床），营销成本难覆盖 → 怎么办？

理解：小件带动大件

大胆假设：用户会因为一件喜欢的小件家具，购买更多配套产品、甚至大件家具。

底层逻辑：用户选择家居用品的终极目标是满足对家居环境氛围的向往，让自己感觉更幸福 → 台灯给了幸福感 → 桌子与台灯不协调 → 换掉桌子 → 最终拥有更满足的家。

连带效应：租房时买小件 → 对宜家产生好感 → 买房后购买大件时优先考虑宜家。

选品策略：卖小件，且是更可能带动大件销售的小件。

放大：从用户分享中选品和扩圈

选品原则：从已有用户分享的品中，选择有明确可放大要素的品。

案例：方格柜卡莱克 → ①搜索量大、笔记数量少（供不应求）②柜子对房间调性重要、容易带动大家具销量。

扩圈方向一-审美风格：用户用深色卡莱克搭配中式风格（配合《繁花》电视剧流行）→ 选择以美学和搭配见长的家居博主 → 实现不错销量。

扩圈方向二-养猫人群：爆火笔记展示猫主子喜欢柜子的封闭多样空间 → 评论区大量用户晒猫 → 测试宠物博主vs养猫的家居博主 → 养猫的家居博主效果更好（能把柜子拍好讲好）→ 内容要求：①展示柜子和收纳筐 ②讲"不是我选中它，而是猫选中了它" ③照片必须有猫。

结果验证：因卡莱克进入宜家的用户，购买了10倍于卡莱克价格的商品 → 营销ROI近20倍 → "小件带动大件"假设得到验证。

宜家的启示
  • 成熟品牌要通过"捕捉-理解"扭转线下惯性，重塑线上选品逻辑。摆脱路径依赖，打破"线下爆品=线上爆品"的惯性思维
  • 保持开放心态，能帮你发现新机会。新的增量也许藏在你从未设想过的用户场景里
  • 成熟品牌可以发挥生态优势，设计流量在品牌生态内的流转路径

案例三：于冬琪商业笔记

种草方法论在内容创作中的应用

背景：2022年才开始做公众号（时机已晚），但成绩不错，多篇文章10万+阅读量。

选题-捕捉：绝大多数选题来自与用户交流 → 用户问的问题再去问其他用户 → 如果大量用户都说"我也遇到了，我也想知道答案" → 碰巧能找到答案 → 就是好选题。

创作-激发：最关注转发率，尤其是创业者的转发（他们是小范围意见领袖，转发会影响团队和合作伙伴）。

问题：选题好坏不稳定、文章质量不稳定、每篇文章需要大量时间投入。

解决方案：调整生产流程，所有步骤都让用户参与：①选题要讲给用户，能被激发、表示强烈好奇才是好选题 ②假设要讲给用户，用户觉得能解决问题、有强烈收获才是好假设 ③大纲和写作前给用户多讲几遍，观察无意识反应（哪里眼睛亮了→讲得好；哪里开始追问→没讲清楚）④写完发布前给不同用户看，反馈哪里有收获、哪里没看懂、哪里没耐心。

结果：文章生产周期变长，但对个人时间精力依赖降低、文章创作更可由团队完成、文章表现更稳定。

充分使用捕捉、理解、放大、激发的工作方式，会帮助你的品牌更容易在小红书上成功。这套方法不仅适用于消费品，只要你的服务或产品是直接面对用户的、面临竞争、需要用户传播，都可以试试。

组织：怎样让种草在企业内发生？

种草型组织的三大特点

掌握方法后，还需要让组织长出相应的种草能力，才能抵消营销的不确定性，保证次次种草都成功。

特点一：种草型员工密度足够高
  • 定义：理解目标用户、对用户需求足够敏感的人，常常本身就是产品用户或超级用户
  • 关键：不是仅集中在一个部门，而是每个部门都要有种草型员工，避免部门间无意义拉扯
特点二：用户反馈成为最高指令
  • 含义：只要说"用户反馈了什么"，就能快速得到关注和支持
  • 对比：有的企业是"老板说"，有的是"财务指标"，种草型企业是"用户反馈"
  • 效果：让种草型员工捕捉到的机会有了落地的可能
特点三：建立支持种草的配套机制
  • 财务机制：支持员工为用户体验投入更多研发成本和时间
  • 考核机制：支持员工花时间接触用户
  • 结果：种草的整个过程高效运转

第一步：提高种草型员工密度

思路一：从内部挖掘

方法1：询问每个人的小红书粉丝数，粉丝数最多的成为小红书负责人。如教育企业找到从大学就运营手帐账号的小博主（1万+粉丝）。

方法2：设计针对性活动让种草型员工"暴露"出来。如影视公司组织"竞猜大奖赛"，预测新作品影响力和财务表现，猜得最准的是客服和财务（都不在内容部门）→ 调到编审部门审剧本、影片、提前给建议 → 内容成功率短时间提升。

核心原理：竞猜题目考验员工的"代入感"，即站在用户角度想问题的能力，这是用户理解能力最直接的体现。

思路二：通过招聘找到

关键：理不理解你的目标用户很重要（不是泛泛的美妆用户，而是具体使用场景的用户）。

案例：美妆公司主打"化妆减负"，但员工都是妆容精致的彩妆高手 → 她们把化妆当探索乐趣，思考怎么叠涂眼影更有层次 → 目标用户把化妆当任务负担，思考怎么5分钟搞定出门 → 定义不同、需求不同，团队无法理解用户麻烦在哪里。

为什么强调目标用户？传统营销大媒体覆盖年代，消费者注意力集中、分层不明显，抓住普遍诉求即可。今天供给过剩、媒体碎片化、消费者决策在具体场景里，必须理解目标用户。

母婴品牌的招聘经验

目标用户：哺乳期妈妈

核心要求："能理解哺乳期妈妈"

两类候选人：①自己本身就是妈妈 ②能理解妈妈的人（一线客服、母婴店金牌导购、妈妈社群主理人、协助育儿工作的人）

面试爸爸：不问"你能不能理解你的妻子"（答案都是能）→ 问"你曾经做过哪些事，来提高对妻子和孩子的陪伴质量？"→ 听具体行动和细节（"我会尽量抽时间"vs"我每周三和周五晚上负责哄睡"）。

三步招聘法：①重新定义招聘标准（把"能理解目标用户"写在标准里）②用行动检验（问"你做了什么"而非"你觉得"）③从细节分辨真假（真正的理解藏在具体行动里）。

第二步：让员工保持理解用户的手感

数码产品公司的三个动作

动作1-招用户当员工：几千名员工中大部分都是自己产品的用户 → 笔试题要求写对产品和行业的理解，筛出用户 → 日常吃饭团建身边坐着的就是真实用户 → 想法被提出时，这些用户会直接用体感投票。

动作2-把用户"写"在流程里：产品立项、运营动作时，文档必须先写清楚：①目标用户是谁 ②要为用户创造的价值是什么 ③具体描述什么时候认识了怎样的用户、如何表达需求 ④论证需求的普遍性 → 如果文档没有这些字段，研发不许接需求。

动作3-创造场域见用户：定期邀请用户来公司 → 访谈者和用户面对面在同一物理空间 → 重点听用户说，不许解释和反驳 → 高管轮流当主持人，员工必须定期参加 → 每个员工心里都有了几个具体鲜活的用户形象 → 会议室、内网、公司墙上贴用户真实照片和评价，让用户"在场"。

三条操作标准
  • 招聘上，坚持把"用户身份"作为加分项
  • 流程上，建立论证用户需求的硬性要求
  • 环境上，创造时刻提醒用户在场的信号

第三步：让用户反馈成为最高指令

三种驱动力
  • 权威驱动：babycare老板要求团队讨论问题时先回到用户，想想见过的用户怎么说、会不会喜欢 → 通过老板权威强行把"用户反馈"放在决策中心 → 对表现好的员工颁发"总裁特别奖"，让团队意识到"用户反馈=机会"
  • 激励驱动：即时奖励比复杂绩效调整更快见效，能迅速让好的行为复制
  • 制度驱动：羊织道给营销部门采购权 → 产品部门做的产品只有拿到营销部门认可才有推向市场的机会和预算 → 产品部门必须重视营销部门传递的用户反馈，必须跟着营销部门接触用户做调研

最终效果：三股动力相互作用，"用户反馈"从口号变成组织常态。用户反馈带来的业务收益直观体现在业绩里，团队习惯了通过关注用户寻求高业绩，重视用户反馈养成习惯。

第四步：建立财务配套机制

阿芙精油的翻译机制

核心问题：种草是长期投入，短期难直接变现，怎样在财务口径下证明种草的经营贡献？

逻辑链条：用户对产品满意 → 衡量指标：NPS（用户净推荐值）→ 用户复购 → 衡量指标：复购率 → 复购率增长带来回报 → 衡量指标：LTV（用户生命周期总价值）

操作方法：把NPS、复购率、LTV全部纳入公司财务测算体系 → 每周带着财务部门一起观察和记录：NPS提升多少 → 对应带来复购率增长多少 → 最终影响LTV多少 → 财务部门积累数据和经验，能算清楚"提升体验的动作虽然短期是成本，但多长时间后能给生意带来多大价值"。

结果：财务核算机制对种草的阻碍被消除，提高了持续种草成功的概率。

两步操作法
  • 第一步-翻译：找到衡量"用户满意度"的核心指标（如NPS），把所有"对用户好"的动作看作是为了提升这个指标
  • 第二步-关联：花时间和耐心记录和验证核心指标与"复购率""LTV"等长期财务回报之间的关联，建立关联模型，让财务能用它评估预算

怎样推动这套变化完成？

小企业路径（船小好调头）
  • 第一步：维持原有业务现金流，释放出自己的精力
  • 第二步：老板亲自带队践行种草方法，带领团队接触用户
  • 第三步：低成本起步，从一个新品、一次营销开始试，看到效果后追加投入
  • 第四步：当种草带来的收入和利润占比越来越高，逐步把重心从原有业务转移过来
大企业路径（部门众多、流程复杂）
  • 第一步：向上管理，拿到授权。争取有足够决策权限的领
