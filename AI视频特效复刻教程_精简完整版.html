第一部分：特效创意挖掘

Higgs Field平台使用

  1. 访问平台 - 浏览特效分类，重点关注运镜类特效（好莱坞级别）
  2. 选择特效 - 评估视觉效果、通用性、复刻难度
  3. 下载视频 - 点击下载按钮保存到本地
⚠️ 重要
只在Higgs Field做创意提取，不在平台生成（需付费）。下载后用于提示词提取和复刻。

特效应用场景

  • 自媒体 - 单镜头创意视频
  • AI短片 - 专业镜头运镜（开场、转场、高潮、结尾）
  • 商业视频 - 产品宣传、品牌广告

第二部分：视频提示词提取

使用Gemini解析视频

  1. 上传视频 - 将下载的特效视频上传到Google Gemini
  2. 输入解析提示词
    解析这条视频，分析出视频提示词，用于AI视频工具生成。 视频特效主题：[从Higgs Field复制的说明] 要求： 1. 详细描述画面视觉元素 2. 分析动态效果和运镜 3. 描述音效和音频（如需要） 4. 给出BGM建议
  3. 获取精简版 - 如果返回太长，继续说："请给出一句完整的精简提示词"
核心原则
• 提示词不是越长越好！过长会让AI混淆重点
• 最佳长度：1-3个关键句子
• 结构：主体 + 动作 + 场景 + 风格

提示词调整要点

用途 调整方法
剪映无音频 删除音效、音乐描述
剪映全能模型 保留完整音视频描述
Veo/Grok 使用英文版本
剪映AI模板 将"一个女人"改为"画面主体"

第三部分：多平台效果复刻

为什么多平台测试？

  • 验证提示词有效性
  • 找到最适合的平台
  • 避免浪费资源
  • 快速迭代优化

Veo 3.1使用

操作步骤

  1. 选择模式：Fast（200积分）或Quality（100积分，音质更好）
  2. 上传参考图（可选）
  3. 输入英文提示词
  4. 设置画面比例（9:16竖屏 / 16:9横屏 / 1:1方形）
  5. 生成并评估效果

优势 音视频同步最佳、英文发音标准、画面质量高

限制 最长8秒、需要付费

Grok使用

特点 每日免费额度、支持参考图

限制 速度较慢、音频效果一般

建议：用于测试阶段

平台选择决策

提取提示词 → 多平台测试 → 效果对比 → 选最优 → 批量生成

第四部分：剪映AI模板制作

完整流程

  1. 打开剪映 - 手机版，点击"开始创作"，选测试图
  2. 进入AI特效 - 选中图片 → AI特效 → 生图/生成视频
  3. 选择模型
    • Seethis Pro（推荐）- 效果好、用户限免
    • 全能生图4.0 - 图片特效
    • 全能音视频 - 自动音效（慢，不推荐）
  4. 输入提示词 - 删除默认词，粘贴你的提示词
    ⚠️ 关键修改
    必须将"一个女人"改为"画面主体"
    这样任何图片都能使用
  5. 设置比例 - 如需要，在提示词末尾加"，画面比例9:16"
  6. 生成检查 - 会生成4个版本，检查效果
  7. 添加音乐（如非全能模型）
    • 点击"音频" → "音乐"
    • 选择合适音乐
    • 剪辑长度匹配视频
    • 添加淡出效果
  8. 导出发布
    • 点击"导出" → "发布模板"
    • 填写标题（简洁有吸引力）
    • 模板说明（清楚说明用法）
    • 封面图：选特效最精彩帧（不要第一帧）
    • 点击"发布"
音频重要性
即使是视觉特效，配上合适音乐会让效果提升10倍！
手动添加音乐比全能模型更快、用户体验更好。

第六部分：图生图+图生视频

什么时候用两步流程？

有些特效需要先改变图片风格，再让图片动起来。

示例：冰糖葫芦特效 = 图生图（变造型）+ 图生视频（摇晃动作）

第一步：图生图

  1. 导入测试图
  2. 进入AI特效 → 生图（选全能生图4.0）
  3. 输入图生图提示词
  4. 设置画面比例（如需要）
  5. 生成并检查4个版本
⚠️ 语言精确性很重要
❌ "头部由草莓组成" → 头会变成草莓形状
✅ "头部四周有草莓" → 周围有草莓装饰

遇到问题时，回顾提示词每个字的含义，找出歧义点。

第二步：图生视频

  1. 选择生成好的图片
  2. 点击"视频生成"
  3. 选择Seethis Pro模型
  4. 输入运动提示词
    示例： 手左右轻轻摇晃，镜头跟随 向前缓慢行走，镜头跟随 原地360度旋转
  5. 设置时长（3/5/8秒）
  6. 生成视频

图生图 vs 直接图生视频

方式 适用场景
图生图+图生视频 风格转换类（冰糖葫芦、雪地人偶）
直接图生视频 动作类（变身、转场、运镜）

第七部分：S平台爆款复刻

为什么必须去水印？

带水印解析会让AI把水印识别为画面元素
提示词会出现"画面左上角有logo跳动"
严重影响复刻效果

去水印操作（抓包法）

  1. 找到视频 - 在S平台找爆款（50万赞+）
  2. 复制链接 - 点击"分享" → "复制链接"
  3. 使用工具 - 粘贴到去水印工具（抓包类）
  4. 下载视频 - 获得100%无痕的原始视频
原理
不是通过AI抹除水印，而是抓取平台原始上传文件
效果：完美去除、原始画质、速度极快

爆款视频解析

  1. 上传无水印视频到Gemini
  2. 使用详细提示词模板
    解析这条视频，分析完整视频提示词，包括： 1. 画面主体和视觉元素 2. 着装、动作、表情 3. 场景环境 4. 运镜方式 5. 旁白内容（逐字记录） 6. BGM风格 7. 音效设计 提供中英文双版本的完整提示词。
  3. 获取中英文提示词
  4. 多平台测试（Veo、Grok、S平台）
  5. 选择最优平台制作
测试结论
教程案例中：Veo 3.1 + 英文提示词 = 最佳效果
• 音视频完美同步
• 发音清晰准确
• 画面质量稳定
• 甚至超越原版（原版无音效）

第八部分：高级技巧

时长控制

核心原则
内容密度必须匹配视频时长！

常见问题

  • 内容过载 - 15秒内容压缩到8秒 → 语速像机关枪
  • 内容截断 - 话说到一半视频就结束了

解决方法

  1. 提前估算 - 自己读一遍旁白，计时，超出就删减
  2. 分段制作 - 拆成2-3个8秒片段，后期拼接
  3. 根据平台选择
    • Veo 3.1（8秒）→ 精简内容
    • S平台（15秒）→ 完整内容
    • 剪映（3/5/8秒）→ 根据选择调整

语言优化

中文 vs 英文

平台/场景 推荐语言
Veo 3.1、Grok、Sora 英文
剪映、国产AI工具 中文
纯视觉特效（无旁白） 都可以

中文发音问题解决

同音字替换法
1. 测试发现某字发音错误
2. 找同音字：如"好"→"豪"、"号"、"毫"
3. 逐个替换测试
4. 找到发音正确的同音字
5. 修改提示词用字

质量提升技巧

1. 提示词添加质量约束

基础版：一个婴儿坐在椅子上说话 优化版：一个婴儿坐在椅子上，用清晰的发音说话，镜头稳定 专业版：一个表情自然的婴儿坐在高脚椅上，用清晰标准的发音说话， 正面特写镜头，专业打光，电影级画质

2. 参考图选择标准

  • 清晰度 - 高清，细节清晰
  • 光线 - 均匀，无阴影
  • 角度 - 正面或3/4侧面
  • 背景 - 简洁干净
  • 主体 - 特征明显，居中

3. 抽卡策略

  • 测试阶段：2-3次，评估稳定性
  • 关键内容：5-10次，选最佳
  • 商业项目：不限次数，追求完美

4. 提示词迭代

初版 → 测试 → 发现问题 → 分析原因 → 优化 → 重测

要点：每次只改一个变量、记录结果、对比测试、80分即可发布

❓ 常见问题

Q1：提示词是不是越长越好？

A：不是！过长会让AI混淆重点。最佳：精简到1-3句核心要素。

Q2：为什么必须去水印？

A：带水印会让AI把水印当画面元素，提示词会出现"画面有logo跳动"，影响复刻。

Q3：中文发音不准怎么办？

A：1)改用英文 2)用同音字替换 3)多次抽卡选最好的

Q4：为什么要多平台测试？

A：验证提示词有效性、发现最优平台、避免浪费、快速迭代。

Q5：封面图为什么不能用第一帧？

A：第一帧通常是原图无特效。应选特效最精彩瞬间（如变身后的效果）。

Q6：为什么手动添加音乐而不用全能模型？

A：1)全能模型很慢，用户体验差 2)消耗积分，普通模型限免。手动添加只需30秒。

Q7：Higgs Field和小云雀有什么区别？

A：Higgs专业级适合进阶；小云雀抖音爆款适合快速跟进。建议结合使用。

Q8：复刻的目的是什么？

A：不只是做模板！还可用于自媒体、AI短片、商业视频、创意库建设。核心是学会完整方法论。

Q9：完全免费能做出好内容吗？

A：可以！小云雀+Gemini+剪映+Grok都有免费额度。但付费工具能大幅提升效率，免费的时间成本往往更贵。

Q10：如何持续获得创作灵感？

A：1)定期浏览Higgs/小云雀/S平台 2)建立创意文档分类整理 3)在现有特效基础上二次创新

📝 十大黄金原则

  1. 提示词不是越长越好 - 精简到核心要素
  2. 必须去水印才能准确解析 - 否则影响提示词质量

  9. 参考图质量决定生成质量 - 高清、正面、光线好
  10. 付费工具提升效率值得 - 免费的时间成本更高
